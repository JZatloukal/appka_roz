Sem patří hlasové nahrávky pro pochvaly po prodeji.

Podporované formáty:
- .mp3
- .m4a
- .wav
- .ogg
- .aac
- .flac

Aplikace soubory načítá automaticky podle názvu souboru.
Po dosažení nastaveného počtu prodejů připraví náhodnou nahrávku a hlídá, aby se stejná nahrávka nepřehrála dvakrát po sobě.
